## Technical Standards

### DMN Structure
- **Hit Policy**: FIRST (sequential rule evaluation)
- **Inputs**: 3-5 variables, camelCase naming
- **Outputs**:
  - `resultado`: "Aprovado" | "Reprovado" | "Pendente"
  - `observacao`: Clinical rationale
  - `fundamentacao`: Guideline reference + evidence level
- **Rule Ordering**: Reprovado → Aprovado → Pendente → Fallback

### Compliance Features
1. **Evidence-Based**: All rules reference international guidelines (ACC/AHA/ESC/HRS/SBC)
2. **Trial References**: Major RCTs cited (SYNTAX, PARTNER, COAPT, FIRE AND ICE, etc.)
3. **Risk Stratification**: SYNTAX score, SCAI stages, CHA2DS2-VASc, RoPE score
4. **Multi-Specialty**: Interventional, imaging, electrophysiology, heart failure, structural

### Key Clinical Algorithms Implemented
- **PCI Decision**: SYNTAX score-based PCI vs CABG
- **Device Therapy**: CRT/ICD selection based on FEVE/QRS/symptoms
- **Valvular**: TAVR vs SAVR risk stratification (STS score, age, frailty)
- **Ablation**: Risk-benefit for AF, VT, SVT procedures
- **Imaging**: Appropriateness criteria matching ACC/SCCT guidelines
- **Anticoagulation**: CHA2DS2-VASc + HAS-BLED balancing
